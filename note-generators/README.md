# Note Generators (Nothing → MIDI)

Plugins that generate MIDI notes/patterns from nothing. These sit at the start of the signal chain.

## Mixed In Key — Captain Series

The Captain plugins generate chord progressions, melodies, basslines, and drum patterns. They output MIDI that you route to instruments.

### Captain Chords (Epic)
Generates chord progressions in any key/scale. Drag-and-drop MIDI output.
- **Best for**: Starting a song, finding progressions for folk/rock/electronic

### Captain Melody (Epic)
Generates melodies that fit over your chord progression.
- **Best for**: Finding melodic ideas, folktronica melodies

### Captain Deep (Epic)
Generates basslines that follow your chord progression.
- **Best for**: Bass patterns for folk rock, alt country, electronic

### Captain Beat (Epic)
Generates drum patterns with various styles and fills.
- **Best for**: Quick drum programming, genre-appropriate beats

### Captain Play (Epic)
Performance/playing tool for live chord triggering.
- **Best for**: Live performance, jamming

## Mixed In Key — Pilot Series (Newer)

### Pilot Chords

### Pilot Melody
- Modern melody generator
### Pilot Bass
- Modern bass generator
### Pilot Arpeggio
- Arpeggio pattern generator

## Workflow Tips for Bitwig

1. Create a Note FX layer in Bitwig
2. Add Captain/Pilot plugin as note generator
3. Route MIDI output to an instrument track
4. Use Bitwig's MIDI routing to send to multiple instruments
5. Layer with Sugar Bytes note FX (Consequence, Thesys) for further transformation
