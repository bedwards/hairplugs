# Artillery2 - YouTube Research Summary

## Plugin Overview
Artillery2 is a real-time effect plugin for stage and studio performance. With over 30 effects triggered via MIDI keyboard, it allows creative sound mangling and manipulation. Effects are mapped to keyboard zones for intuitive live performance.

## Key Insights

### Setup and Routing
- Works as an insert effect plugin (VST/AU)
- Route audio through Artillery by placing it on an audio channel
- In Ableton Live: drag plugin onto audio channel with clip playing
- All sound from the clip routes through Artillery for processing

### Keyboard Zone System
- MIDI keyboard keys are mapped to different effects
- Click and drag above the keyboard to create zones
- Each zone can have a different effect assigned
- Zones can be resized and repositioned
- Effects are triggered by playing MIDI notes in the corresponding zone

### Effects Categories
- Over 30 effects available
- Organized into categories (e.g., "granular")
- Include Looper, filters, delays, distortion, and more
- Effects can be layered across multiple zones

### Looper Effect (Granular Category)
- Keyboard input quantization (e.g., eighth notes) for synced triggering
- Loop is triggered in sync with incoming audio even with imprecise timing
- Loop size is adjustable
- Great for creating rhythmic variations on the fly

### Global Presets
- "Empty" preset provides a blank starting point
- Global presets save entire keyboard mappings
- Individual effect presets also available

### Performance Features
- Designed for real-time live performance
- MIDI keyboard-driven effect triggering
- Intuitive mapping makes it stage-ready
- Can also be used creatively in studio production

## Creative Uses
- Live performance effect manipulation via MIDI keyboard
- Creating glitch and stutter effects in real-time
- Building complex effect chains across keyboard zones
- DJ-style sound manipulation in the studio
