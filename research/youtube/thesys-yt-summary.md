# Thesys - YouTube Research Summary

## Plugin Overview
Thesys is a step sequencer by Sugar Bytes for building patterns, generating pitch sequences, and driving both its internal synthesizer and external instruments via MIDI. It functions as both a composition tool and a performance instrument.

## Key Insights

### Pattern Building Basics
- Start with the default preset (user folder) for a clean slate
- Individual sequencers can be reset via reset buttons
- Start button begins playback
- Internal synthesizer for standalone use, or route MIDI output externally
- Set root note and scale type (e.g., D Major) to constrain patterns to key

### DAW Integration & Triggering
- Three trigger modes:
  1. **Host** - Starts/stops with DAW transport
  2. **Note Toggle** - Starts on first MIDI note, stops on next
  3. **Note On/Off** - Plays while MIDI note is held
- Works with external MIDI keyboards/controllers
- Can drive any external synthesizer via MIDI output
- MIDI export for capturing patterns into the DAW

### Performance Sequencer
- **Octave Transposition Row** - Shift notes up/down 1-2 octaves per step
- **Mode Sequencer** - Controls chord/note playing behavior
- **Velocity Row** - Controls dynamics per step
- Shift-click and drag to edit series of steps
- Structures patterns melodically and rhythmically

### Generating Pitch Sequences
- Built-in pitch generation tools
- Scale-aware pattern generation
- Musical results constrained to selected key and scale
- Quick way to create varied melodic content

### Making Patterns More Varied
- Performance sequencer is the key to variation
- Octave transposition adds register variety
- Combine multiple sequencer rows for complex patterns
- Small changes create significant musical impact

### MIDI Export
- Capture generated patterns as MIDI in your DAW
- Use Thesys for idea generation, then edit in DAW
- Bridge between generative sequencing and traditional composition

## Creative Uses
- Generating melodic and rhythmic sequences
- Driving external synthesizers with complex MIDI patterns
- Live performance with MIDI keyboard triggering
- Creating evolving patterns with performance sequencer
- Composing scale-aware melodies and arpeggios
- Exporting MIDI ideas for further arrangement in DAW
