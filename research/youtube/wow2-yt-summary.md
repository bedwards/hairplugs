# WOW2 - YouTube Research Summary

## Plugin Overview
WOW2 is a creative multi-filter box by Sugar Bytes featuring 21 filter types, a famous vowel mode, extensive distortion options, and deep modulation routing. Known for its polished UI and warm sound character. Used extensively in dubstep and electronic music (notably associated with Skrillex's sound).

## Key Features and Capabilities

### 21 Filter Types
- **Low Pass**: multiple models including Sugar Bytes own (O3O), Moog-modeled, diode filter
- **High Pass**: 2-pole and 4-pole options
- **Band Pass**: adjustable width via resonance
- **Special Modes**: mid boost, peak, notch, band reject, comb filters
- Pole settings (2, 4, 8) control steepness of cutoff slope
- SAT (saturation) modes maintain gain when increasing resonance
- Visual cutoff display in center of main knob shows filter behavior in real-time

### Resonance
- Hard to push into harsh self-oscillation (well-designed gain handling)
- Much smoother than typical stock DAW filters
- Enhances vowel sounds at higher settings
- Visual feedback in the cutoff display

### Vowel Mode
- Activated via mouth icon button
- Morphs between two selectable vowel sounds (left and right)
- Cutoff knob sweeps between the two vowels
- Especially effective with band pass filter and higher resonance
- Creates the "talking bass" sound popular in dubstep

### Distortion Engine
- 7 distortion types: parabolic (tube-like), hyperbolic (aggressive), diabolic (extreme), sine (wave shaper), one-bit, crush (bit reducer), digitize (sample-and-hold)
- Pre-filter or post-filter placement drastically changes character
- Pre-filter: distorts raw audio, then filter shapes the result
- Post-filter: filter shapes audio first, then distortion is applied
- Sine distortion retains bass and adds thunderous low end
- Bit crusher and digitize for lo-fi and 8-bit effects

### Modulation Section
- **Envelope Follower**: reacts to input audio dynamics
  - Attack and decay controls
  - Side-chain frequency selector (listen to lows, highs, or full signal)
  - Input or output source selection
  - Modulation destinations: cutoff, drive, volume, and more
  - Positive or negative modulation direction
- LFO and sequencer modulation modes
- Multiple simultaneous modulation destinations
- Visual feedback shows modulation movement on controls

### Core Controls
- Cutoff knob with visual frequency display
- Resonance with smooth behavior
- Volume (output level)
- Dry/wet mix for subtle effects
- Randomize button for quick preset discovery

## Workflow Tips and Tricks
- Use the center display of the cutoff knob for quick visual reference
- Pre vs. post distortion placement is a key creative decision
- Envelope follower with kick-focused side-chain creates pumping compression
- Modulate distortion drive downward on transients for dynamic processing
- Combine envelope follower on volume + distortion for complex pumping
- Start with presets organized in folders, then customize from there
- Randomize button generates unexpected starting points
- SAT mode prevents volume loss when pushing resonance high
- Comb filter + resonance creates tube-like resonant sounds

## Creative Uses
- Classic filter sweeps (low pass, high pass) for electronic music builds
- Dubstep "talking bass" using vowel mode with band pass
- Rhythmic filter patterns via modulation presets
- Compression-like effects using envelope follower on volume
- Extreme sound destruction with diabolic distortion and 1-bit modes
- Lo-fi and retro effects with bit crusher and digitize
- Dynamic distortion that reacts to input signal
- Cinematic sound design with special filter modes (comb, notch)
- Drum processing with pre/post distortion and envelope following
