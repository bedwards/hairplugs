# WOW2 - YouTube Research Summary

## Plugin Overview
WOW2 is a creative multi-filter box by Sugar Bytes. It provides various filter types (high-pass, low-pass, band-pass, vowel mode) with deep modulation capabilities. Commonly used in electronic music for filter sweeps, vowel effects, and dynamic tonal shaping.

## Key Insights

### Filter Types
- **High-pass filter** - Removes low frequencies
- **Low-pass filter** - Removes high frequencies
- **Two-pole variants** - Different filter slopes available
- **Vowel mode** - Creates vocal-like formant filter effects
- Commonly used filter types in electronic music, made creative here

### Basic Parameters (5 Main Knobs)
- Core controls for filter cutoff, resonance, and character
- Intuitive layout for quick sound shaping
- Each knob has significant impact on the output
- Part 1 tutorial covers these fundamentals

### Vowel Mode
- Creates vocal "wah" and formant sounds
- Separate tutorial dedicated to this feature (Part 2)
- Unique selling point of WOW2
- Popular for creating talking/singing filter effects

### Modulation Section
- Deep modulation capabilities below the main filter section
- LFOs, envelopes, and step sequencers for filter automation
- Creates dynamic, evolving filter movements
- Sonic Academy tutorial covers filter section + modulation in two parts

### Practical Usage
- Insert on any instrument or audio track
- Quick to set up - just select filter type and turn knobs
- Can be used as a simple filter or as a creative effect
- Demonstrated on various audio sources

### Multiple Tutorial Perspectives
- EDMProd: Technical overview of parameters (recommended reading their full review)
- Audio Plugin Deals: Quick demo of basic high/low pass usage
- Sonic Academy (Aiyn Zahev): Comprehensive two-part breakdown (filter + modulation)
- The Sonic Academy video is the most thorough at 10+ minutes

### Filter in Mix Context
- Demonstrated on instruments within full arrangements
- Shows how WOW2 can be applied to different parts of a mix
- Different filter settings suit different instruments and genres

## Creative Uses
- Filter sweeps and buildups in electronic music
- Vowel/formant effects for vocal-like filter sounds
- Dynamic filter modulation synced to tempo
- Creating evolving filter textures on pads and synths
- DJ-style filter effects in production
- Adding movement to static sounds through modulated filtering
