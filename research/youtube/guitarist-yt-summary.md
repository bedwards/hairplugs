# Guitarist - YouTube Research Summary

## Plugin Overview
Guitarist is a complete guitar playing application by Sugar Bytes, available as standalone and as a VST/AU plugin. It includes amplifiers, effects, three electric guitars, and a step-sequencing system that replaces a live guitar player with an intuitive programmable interface.

## Key Insights

### Guitar Models
- Three electric guitar models included: Duesenberg Star Player, Telly (Telecaster-style), and Strat (Stratocaster-style)
- Guitar choice is stored and recalled by host DAW
- Each guitar has its own tonal character
- Settings panel for guitar selection

### Step Sequencing System
- Pattern-based sequencer drives guitar performance
- Choose which string to play at any time
- Playing styles make it easy to create realistic guitar parts
- Step resolution adjustable (e.g., 16 steps = one bar at 16th notes)

### Chord System
- Chord sequencer works alongside pattern sequencer
- Chord board for selecting chord voicings
- Start with default preset (user folder) for empty workspace
- Create triggers on steps, then assign chords
- Chord sequencer runs at same speed as pattern sequencer
- "Rep" (repeat) function for sustained chords

### Amplifiers and Effects
- Built-in guitar amplifier modeling
- Effects processing chain included
- Authentic guitar tone shaping
- Integrated into the plugin for all-in-one guitar tracks

### Deep Programmability
- Intuitive interface despite deep parameter control
- "Although tasteful guitar playing is a thing of beauty, this instrument still has a lot to offer"
- Used on multiple professional projects
- Works as both creative tool and production shortcut

### Workflow
1. Choose guitar model
2. Set up chord progression in chord sequencer
3. Program picking/strumming patterns in step sequencer
4. Apply amp and effects for tone
5. Automate or perform in real-time

## Creative Uses
- Adding realistic guitar parts without a guitarist
- Programming complex strumming and picking patterns
- Creating guitar accompaniments for electronic music
- Quick guitar sketch ideas during songwriting
- Building guitar riffs with step-sequenced precision
