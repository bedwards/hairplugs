# Turnado - YouTube Research Summary

## Plugin Overview
Turnado is a multi-effect plugin by Sugar Bytes designed for massive real-time audio manipulation. It features 8 main effect knobs for immediate hands-on control, a Dictator for single-knob macro control, and comprehensive MIDI mapping for live performance and studio use.

## Key Insights

### 8 Main Knobs
- Each knob controls a different effect
- Designed for real-time hands-on manipulation
- Turn a knob up to engage the effect
- Multiple effects can be combined simultaneously
- Core interaction model: twist knobs for instant transformation

### The Dictator
- Single macro knob that controls all 8 effect knobs at once
- Maps custom curves for each knob's response to the Dictator
- One-knob control over complex multi-effect transitions
- Perfect for live performance where you need single-hand control
- Dedicated tutorial on Dictator setup and use

### MIDI Control
- All 8 main knobs mapped to MIDI CC by default
- All controls in the Edit Page can be MIDI-controlled
- Set up as MIDI-controlled effect in DAW for MIDI routing
- Use MIDI controller or MIDI track data for automation
- "Getting hands-on with the main knobs is what the plug-in is all about"

### Ableton Live Integration (Joshua Casper tip)
- Map Turnado knobs to Ableton macro controls
- Control Turnado without opening the plugin window
- Keeps workflow streamlined in existing session layout
- Works great with pre-built presets
- Useful for both live and studio performance

### Practical Sound Design
- Demonstrated transforming bass lines and chord progressions
- Shows before/after of processed audio
- Multiple effects stacking creates dramatic transformations
- Great presets included as starting points
- Effects can be subtle or extreme

### Edit Page
- Deeper parameter control for each effect
- Fine-tune effect behavior beyond the main knobs
- All Edit Page parameters also MIDI-controllable
- Access detailed settings without losing immediacy of main knobs

## Creative Uses
- Live performance audio mangling with hands-on control
- Transforming bass lines and melodic content in real-time
- Building complex effect transitions with the Dictator
- Studio sound design with MIDI automation
- Creating dramatic drops and builds for electronic music
- DJ-style audio manipulation in production context
