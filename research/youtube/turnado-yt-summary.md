# Turnado - YouTube Research Summary

## Plugin Overview
Turnado is a real-time multi-effect processor by Sugar Bytes with 8 macro knobs controlling a vast array of effects. Features a built-in automation system called "The Dictator" for sequencing effect changes. Designed for both live performance and studio production.

## Key Features and Capabilities

### 8 Macro Knob System
- 8 main knobs, each controlling a different effect
- Effects include: filters, guitar amps, phasers, loopers, stutters/mutes, freeze reverb, granulizer, and more
- Each knob slot can be swapped to any available effect type
- Turning a knob up engages and intensifies that effect
- Preset sets provide curated combinations of 8 effects

### The Dictator (Built-in Automation)
- Sequencer-style automation system for all 8 knobs simultaneously
- Controlled by a single master slider (assignable to MIDI controller)
- Place automation points along 8 lanes to define knob values at each stage
- Two points create a glide/transition between values
- Shift+drag to adjust point values; colored shadows show intensity
- Dice button randomizes lanes; X button clears them
- Preset system with save/load for Dictator configurations
- Padlock button to keep Dictator settings when changing Turnado presets
- Can be linked to an LFO controller for automatic sweeping

### MIDI Control
- Default mapping: 8 main knobs to MIDI CC 1-8
- Right-click learn function to reassign any MIDI controller
- All Edit Page controls can also be MIDI mapped
- Record MIDI control as clips in DAW for arrangement use

### Effect Types Available
- Filters (various types including vowel filter)
- Guitar amp simulation
- Phasers
- Loopers with tempo-synced timing
- Stutter/mute effects (editable presets)
- Freeze reverb
- Granulizer
- Slice arrangers
- Delays and distortion

### DAW Automation
- Standard DAW automation can control individual knobs
- Precise control over effect timing and intensity
- Loop effects on master channel for tempo-based repeats

## Workflow Tips and Tricks

### Ableton Live Integration (Joshua Casper method)
1. Open Configure window on Turnado device
2. Click each of the 8 macro knobs to register parameters
3. Close Turnado, disable Configure
4. Group device into Audio Effect Rack (Ctrl+G)
5. Map 8 configured parameters to 8 rack macro knobs
- Result: control Turnado without opening the plugin window
- Changing effect presets inside Turnado preserves macro mapping
- Create multiple rack saves with different Turnado configurations

### General Tips
- Use preset effect sets as starting points
- Random button for quick experimentation
- Edit button on individual effects for fine-tuning stutter timing etc.
- Combine Dictator with DAW automation for layered control
- Effects that would take a long time manually can be built in minutes

## Creative Uses
- Live performance real-time effects processing
- Studio production: intricate effect sequences on bass, chords, drums
- Creating complex-sounding effect chains from simple source material
- DJ-style transitions and builds
- Automating filter sweeps, stutters, and glitch effects across arrangements
- Master bus effects processing with tempo-synced loops
