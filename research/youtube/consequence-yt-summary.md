# Consequence - YouTube Research Summary

## Plugin Overview
Consequence is a chord sequencer plugin by Sugar Bytes that generates melodic and harmonic patterns. It combines a chord sequencer, gate sequencer, modulation sequencer, and performance sequencer into a powerful composition tool.

## Key Insights

### Sequencer Architecture (4 layers)
1. **Chord Sequencer** - Determines which notes/chords are played at each step
2. **Gate Sequencer** - Triggers notes; the most basic sequencing element
3. **Performance Sequencer** - Handles articulation, register, and rhythm
4. **Modulation Sequencer** - Controls parameters in instrument, filter, and effects modules

### Preset System
- Can combine presets from different sections for thousands of possibilities
- Presets for individual sequencer sections can be mixed and matched
- Start button toggles playback (can also sync to host clock)
- No tweaking required to get interesting results from preset combinations

### Pattern Building
- Click and drag to create/edit sequencer steps
- Right-click to reset steps
- Shift-click and drag to edit a series of steps at once
- Each sequencer section has independent pattern length and direction

### Performance Sequencer Features
- **Octave Transposition** - Shift notes up/down 1-2 octaves per step
- **Mode Sequencer** - Determines how chords are played (strum, arpeggio, etc.)
- **Velocity Sequencer** - Controls dynamics per step
- Structures patterns both melodically and rhythmically

### MIDI Output
- Can route MIDI to any external instrument or synth
- Also has an internal synthesizer for standalone use
- MIDI export capability for capturing generated patterns

### Scale and Key Control
- Set root note and scale type (major, minor, etc.)
- All generated patterns conform to the selected scale
- Easy to stay in key while experimenting

## Creative Uses
- Generating chord progressions and melodic ideas
- Creating evolving harmonic sequences for electronic music
- Composing arpeggiated patterns with precise control
- Using as a composition/sketching tool to overcome writer's block
- Driving external synthesizers with complex MIDI patterns
