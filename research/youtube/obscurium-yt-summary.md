# Obscurium - YouTube Research Summary

## Plugin Overview
Obscurium is a generative synthesizer by Sugar Bytes built around a 32-step motion sequencer with 16 flexible parameters. It generates scales, chords, arpeggios and combines FM and analog synthesis. Creates organic, mysterious sound textures, chord sequences, and even drums.

## Key Features and Capabilities

### Motion Sequencer (Core)
- 32-step sequencer with 16 flexible parameters displayed as colored dots
- Parameters control oscillators, filters, pitch, chords, polyphony, and more
- Each parameter has four sub-options: envelope amount, LFO amount, min, and max range
- Arrow toggle per parameter to enable/disable motion sequencer control
- Loop bars to constrain playback to specific step ranges

### Dual Oscillator Engine
- Analog oscillator: saw, pulse (with pulse width), super saw, sync frequency
- FM oscillator: three routing algorithms, ratio knob, harmonic button
- Mix parameter crossfades between analog and FM per step
- Deep sound design from pads to percussive hits

### Pitch and Chord System
- Scale selection from many scale types (Dorian, Major, etc.)
- 54 chord tables derived from selected scale
- Polyphony control (1-8 voices) per step
- Arpeggiator (ARP) parameter with three octave range
- Pitch offset, octave shift, detune, and glide between notes

### Draw Tools
- **Pin Tool**: free draw, adjustable step count (1-32 steps at once)
- **Line Tool**: straight lines at user-defined angles
- **Paint Brush**: random generator with adjustable intensity
- **Sine Tool**: draws sine waves with frequency and amplitude control
- **Draw All**: draws all parameters at once with adjustable offset/spread

### Modifiers
- Move: shifts all parameter values up/down (mirrors at boundaries)
- Stretch/Shrink: compresses or expands values toward/away from center
- Rotate: cycles parameter values through neighboring lanes
- Copy/paste, swap left/right for individual steps
- Lock button: isolates a step for repeated audition
- ALL toggle: apply modifiers to single parameter or all at once

### Morph and Shift
- A/B state system: every preset has two states
- Morph slider blends between A and B states
- Shift slider rotates parameter assignments (cutoff becomes noise, etc.)
- Both sliders can be modulated by LFO, envelope, or step sequencer
- Smooth ramping option for shift transitions

### Super Obscure Mode
- Full random generator for all 16 parameter lanes
- Grid slider controls randomization granularity (1-32 steps)
- Infinity mode for continuous re-randomization
- Multiplier controls vertical spread, Add controls position
- Random Add with adjustable intensity

### Clock System
- Step speed from whole notes to very fast divisions
- Swing control for groove
- Direction modes: forward, backward, ping-pong, random, skip patterns

## Workflow Tips and Tricks
- Right-click keyboard to lock a note for hands-free sound design
- Copy A state to B, then modify B for meaningful morph transitions
- Use loop bars to isolate good random segments, then copy/paste
- Turn off parameter arrows to exclude them from shift/morph
- Combine envelope and mod lane for rhythmic amplitude patterns
- Use mods parameter to modulate the modulators (LFO speed per step)

## Creative Uses
- Generative evolving soundscapes and ambient textures
- Rhythmic chord progressions with per-step harmony changes
- Percussive loop design using FM synthesis and short envelopes
- Sound design exploration through random generation and morphing
- Live performance with real-time morph/shift modulation
- Creating organic, unpredictable melodic sequences
