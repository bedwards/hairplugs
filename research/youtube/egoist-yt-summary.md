# Egoist - YouTube Research Summary

## Plugin Overview
Egoist is a groove machine by Sugar Bytes that combines a sample slicer, bass synthesizer, and drum machine, each with its own effect sequencer. It is designed as a creative composition and track-sketching tool.

## Key Insights

### Three Engines
1. **Slicer** - Loads, slices, and rearranges audio samples
2. **Beat Section** - Built-in drum machine
3. **Bass Section** - Synthesized bass lines
- Each engine can be independently routed through an effect sequencer

### The Slicer
- Load any audio sample and it gets automatically sliced
- Rearrange slices to create entirely new patterns
- "Play" button lets you hear the original unsliced sample for reference
- Slicing and rearranging can create music "you never even imagined" from the source
- Drag and drop sample loading

### Slice Rearrangement
- Visual step sequencer shows slice order
- Click to assign different slices to different steps
- Can reverse, pitch-shift, and manipulate individual slices
- Creates completely transformed versions of source material

### Effect Sequencer
- Each of the three engines has its own effect sequencer
- Step-based effect programming
- Effects change per step for dynamic, evolving patterns
- Combines the sound engines with effect processing

### Content Library
- 400+ sounds in 16 folders (drums, guitar, modular, SFX, etc.)
- 1,000+ sampled drum sounds for the drum machine
- Extensive preset library for quick starting points

### Workflow
- Load a sample into the slicer
- Slice it up and rearrange for melodic/rhythmic content
- Add a beat from the drum machine
- Layer in bass from the bass section
- Route through effect sequencers for finishing touches

## Creative Uses
- Quick track sketching and idea generation
- Creating grooves from any audio sample
- Combining sliced audio with synthesized bass and drums
- Beat-making with integrated sample manipulation
- Live performance groove generation
