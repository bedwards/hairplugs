# Folk / Acoustic / New Folk / Modern Folk

## Core Sound
Acoustic guitar, banjo, mandolin as primary instruments. Clean, natural, intimate.

## Quick Start Chains

### Simple Acoustic Guitar
```
Kontakt (Session Guitarist - Picked Acoustic) → soothe2 → Valhalla Room
```
Or play live acoustic through:
```
[Audio In] → soothe2 (tame resonances) → Oxford Inflator (subtle warmth) → Valhalla Room (send, small room)
```

### Banjo + Mandolin Texture
```
Kontakt (8Dio Mandolin Strummer / RealiBanjo) → bx_greenscreamer (very subtle) → Comeback Kid (slapback) → Valhalla Room
```

### Folk Fingerpicking with Guitarist
```
Guitarist (Picking category) → WOW2 (subtle, cutoff high) → Comeback Kid (short slapback)
```
Guitarist settings: amp_drive low, model_x_tone bright, chorus off

### Atmospheric Modern Folk
```
Captain Chords → Consequence (gentle arp) → Aparillo (pad preset) → soothe2 → Valhalla Room (long decay)
Kontakt (Picked Acoustic) → Comeback Kid → Valhalla Room
```
Layer the atmospheric synth pad underneath the acoustic guitar.

- Streetly Tapes strings for folk ballad sections
- Flute tapes for melodic counterpoint
- Choir tapes for atmospheric sections
