# Alt Country

## Core Sound
Electric guitar, acoustic guitar, pedal steel, fiddle. Country-influenced but with indie/alternative edge.

## Quick Start Chains

### Twangy Electric Guitar
```
Guitarist (Rock or Country patterns) → bx_greenscreamer → WOW2 (subtle auto-wah) → Comeback Kid (slapback) → Valhalla Room
```

### Pedal Steel
```
Kontakt (Pedal Steel library) → soothe2 → Comeback Kid (long delay) → Valhalla Room (hall)
```

### Fiddle
```
Kontakt (The Fiddle / Misfit Fiddle) → soothe2 → Valhalla Room
```

### Country Drums
```
Captain Beat → Kontakt (RealiDrums) → DynOne3 → Valhalla Room (tight room, send)
```

### Full Alt Country Setup
```
Track 1: Guitarist (Picking/Indie) — rhythm guitar
Track 2: Kontakt (Electric Sunburst Deluxe) — lead guitar
Track 3: Kontakt (Pedal Steel) — atmosphere
Track 4: Kontakt (Scarbee Bass) — bass
Track 5: DrumComputer or RealiDrums — drums
Track 6: Kontakt (The Fiddle) — fiddle fills
```

### Honky-Tonk Piano
```
Kontakt (any piano library) → bx_greenscreamer (barely on) → Comeback Kid (slapback)
```
