# Folktronica

## Core Sound
Folk instruments blended with electronic production. Acoustic meets digital. Glitchy textures over warm organic sounds.

## Quick Start Chains

### Glitched Acoustic Guitar
```
[Audio In / Kontakt Picked Acoustic] → Effectrix (stutters, filters) → Comeback Kid → Valhalla Room
```
Or:
```
[Audio In] → Looperator (rhythmic slicing) → WOW2 (vowel filter) → Valhalla Room
```

### Electronic Underbelly
```
Pilot Chords → Consequence (arp) → Factory (pad) → soothe2 → Valhalla Room (long hall)
```

### Folktronica Beat
```
Captain Beat → DrumComputer → Effectrix (selective glitches) → DynOne3
```
Layer with:
```
[Acoustic percussion / Kontakt Misfit Stompbox] → Egoist (slice & rearrange) → Comeback Kid
```

### Texture Layer
```
Pilot Melody → Thesys (transform notes) → Obscurium (spectral synth) → Turnado (subtle) → Valhalla Room
```

### Banjo Meets Electronics
```
Kontakt (RealiBanjo) → Permut8 (granular glitch, subtle) → WOW2 → Comeback Kid
```

### Full Folktronica Template
```
Track 1: Kontakt Picked Acoustic → soothe2 → Room verb [DRY ANCHOR]
Track 2: Same acoustic → Effectrix → Comeback Kid [GLITCHED MIRROR]
Track 3: Captain Chords → Factory (pad) → Valhalla Room [ELECTRONIC BED]
Track 4: Captain Beat → DrumComputer → Egoist [BEATS]
Track 5: Pilot Melody → Obscurium → Turnado → Hall verb [TEXTURE]
Track 6: M-Tron Pro (tape sounds) → WOW2 → Comeback Kid [VINTAGE ELECTRONIC]
```

## Key Technique
The magic is in the contrast: keep one "dry anchor" track (clean acoustic) while everything else gets processed. The listener hears folk but feels electronic.
