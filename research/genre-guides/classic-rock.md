# Classic Rock (with Hammond Organ)

## Core Sound
Electric guitar, Hammond organ, bass, drums. Warm, analog, driven.

## Quick Start Chains

### Hammond Organ
```
M-Tron Pro IV (organ tapes) → bx_greenscreamer (overdrive) → Valhalla Room (spring-like short decay)
```
M-Tron Pro settings: tone warm, tape_rewind for Leslie-like effect

Or use Kontakt with WURL-e Studio for Wurlitzer.

### Overdriven Guitar
```
Guitarist (Rock patterns) → bx_greenscreamer (full drive) → Decapitator (saturation) → Comeback Kid (slapback) → Valhalla Room
```

### Bass
```
Kontakt (Scarbee Rickenbacker Bass) → Decapitator (light tube saturation) → DynOne3
```

### Drums
```
Captain Beat → Kontakt (RealiDrums) → Decapitator (subtle tape sat) → DynOne3 → Valhalla Room (tight room, send)
```

### Full Classic Rock Template
```
Track 1: Guitarist (Rock) → bx_greenscreamer → Room verb [RHYTHM GUITAR]
Track 2: Kontakt (Electric Sunburst) → Decapitator → Comeback Kid [LEAD GUITAR]
Track 3: M-Tron Pro (organ) → bx_greenscreamer → Room verb [ORGAN]
Track 4: Kontakt (Scarbee Bass) → Decapitator → DynOne3 [BASS]
Track 5: Captain Beat → RealiDrums → Room verb [DRUMS]
```

## Mix Bus
```
Decapitator (very subtle, tape mode) → DynOne3 → Oxford Inflator → Insight 2
```
The Decapitator on the bus adds classic analog glue.
