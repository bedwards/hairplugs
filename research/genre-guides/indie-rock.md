# Indie Rock (Electric Guitar + Electronic Elements)

## Core Sound
Electric guitar meeting electronic production. Lo-fi elements, creative effects, textural layers.

## Quick Start Chains

### Indie Guitar Texture
```
Guitarist (Indie patterns) → WOW2 (subtle filter movement) → Comeback Kid (dotted 8th delay) → Valhalla Room
```

### Glitchy Guitar
```
Guitarist → Effectrix (selective stutters) → Comeback Kid → Valhalla Room
```

### Synth Pad Layer
```
Pilot Chords → Unique (big unison pad) → soothe2 → Valhalla Room (long decay)
```

### Electronic Drums
```
Captain Beat → DrumComputer → Looperator (add variation) → DynOne3 → Room verb
```

### Bass (Synth)
```
Captain Deep → Cyclop → DynOne3
```

### Atmospheric Vocal Synth
```
Synthesizer V Studio 2 → Effectrix → Valhalla Room (huge hall)
```

### Full Indie Rock Template
```
Track 1: Guitarist (Indie) → WOW2 → Comeback Kid [GUITAR TEXTURE]
Track 2: Kontakt (Electric Sunburst) → bx_greenscreamer → Room verb [GUITAR LEAD]
Track 3: Unique (pad) → soothe2 → Hall verb [SYNTH BED]
Track 4: Captain Beat → DrumComputer → DynOne3 [DRUMS]
Track 5: Cyclop → DynOne3 [BASS]
Track 6: Aparillo → Turnado → Hall verb [ATMOSPHERE]
```

## Key Indie Techniques

### Shoegaze Wall
```
Guitarist → bx_greenscreamer (full) → WOW2 (slow LFO) → Valhalla Room (huge decay, high mix)
```

### Post-Rock Build
Start with clean Guitarist picking, automate bx_greenscreamer drive from 0 to full over 16 bars, bring in Turnado dictator knobs gradually.
