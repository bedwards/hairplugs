# Electronic / Chill / Experimental

## Core Sound
Pure electronic. Synthesizers, programmed drums, ambient textures. Chill vibes and experimental sound design.

## Quick Start Chains

### Chill Pad
```
Pilot Chords → Aparillo (Ambience preset) → soothe2 → Valhalla Room (long decay, high mix)
```

### Ambient Texture
```
Obscurium (explore the XY pad) → Turnado (very subtle movements) → Valhalla Room (massive decay)
```

### Chill Beat
```
Captain Beat → DrumComputer → WOW2 (filter sweeps) → Comeback Kid (synced delay) → Valhalla Room (send)
```

## Experimental Techniques

### Feedback Loop
In Bitwig, create a feedback routing:
```
[Audio] → Turnado → [Send to self with gain < 1.0]
```
Turnado's 8 dictator knobs make this controllable chaos.

### Granular from Acoustic
Record acoustic guitar, then:
```
[Recorded Audio] → Egoist (extreme slice) → Valhalla Room
```
Turns folk recordings into electronic textures.
