# Folk Rock

## Core Sound
Acoustic guitar foundation + drums + electric bass. Energy and drive while keeping folk roots.

## Quick Start Chains

### Driving Acoustic Guitar
```
Kontakt (Strummed Acoustic 2) → bx_greenscreamer (light overdrive) → DynOne3 (compress) → Comeback Kid → Valhalla Room
```

### Electric Bass
```
Kontakt (Scarbee Rickenbacker Bass) → Turnado (subtle, dictator 1 = compression) → DynOne3
```

### Drum Track
```
Captain Beat → DrumComputer (or Microtonic) → DynOne3 → Valhalla Room (send, tight room)
```
Or:
```
Captain Beat → Kontakt (RealiDrums) → DynOne3 → Valhalla Room (send)
```

### Full Band Setup
```
Track 1: Kontakt Strummed Acoustic → soothe2 → Valhalla Room (send)
Track 2: Kontakt Rickenbacker Bass → DynOne3
Track 3: Captain Beat → DrumComputer → DynOne3 → Room verb (send)
Track 4: M-Tron Pro (strings or flute) → Valhalla Room (send, hall)
```

## Mix Bus
```
soothe2 → DynOne3 (gentle glue) → Oxford Inflator → Insight 2
```
